## To Compile
Use the make file

```bash
  mingw32-make
```

#### OR
Run the command

```bash
  g++ -I src/include -L src/lib -o EcoSystemSimulator *.cpp src/userIncludes/*.cpp -lmingw32 -lSDL2main -lSDL2 -lSDL2_image -static-libstdc++
```
